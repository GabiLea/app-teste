Iniciando o projeto

Tenha instalado em sua maquina:
 - Node v6.12.1
 - Angular/cli v8.3
Abra o projeto no Visual studio Code e abra um terminal e instale os pacotes do node
 npm install
 
Agora abra dois terminais.
Em um deles digite:
 json-server --watch db.json

No outro digite:
 ng serve

Por fim acesse:
http://localhost:4200/
